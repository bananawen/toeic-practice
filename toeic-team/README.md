# TOEIC 專案 Agent 團隊

## 團隊成員

### 1. 前端工程師 (Frontend Engineer)
- **職責**: 網頁開發、UI 實現、功能優化
- **技能**: HTML, CSS, JavaScript, TTS 整合
- **檔案**: toeic-practice.html

### 2. 後端工程師 (Backend Engineer)
- **職責**: API 整合、資料處理、資料庫設計
- **技能**: Python, Node.js, API 開發
- **負責**: Notion API、MiniMax TTS API

### 3. 美術工程師 (UI/UX Designer)
- **職責**: 網頁美化、使用者體驗優化
- **技能**: CSS 設計、動畫、品牌視覺

### 4. 測試工程師 (QA Engineer)
- **職責**: 功能測試、BUG 回報、品質把關
- **技能**: 測試用例撰寫、自動化測試

## 協作流程

1. **需求提出** → 由主 Agent 統籌
2. **開發實作** → 前端/後端分工
3. **設計優化** → 美術工程師美化
4. **測試驗收** → 測試工程師把關
