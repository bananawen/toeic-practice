# TOEIC 練習網站 - 專案架構

## 技術棧

### 前端
- **框架**: React + Next.js (App Router)
- **語言**: TypeScript
- **樣式**: Tailwind CSS
- **PWA**: 可安裝應用、離線存取
- **部署**: Vercel

### 後端
- **框架**: FastAPI (Python) + Next.js API Routes
- **語言**: TypeScript + Python
- **資料庫**: PostgreSQL
- **AI 整合**: MiniMax TTS, ElevenLabs

### 跨平台
- 響應式設計 (Responsive Web Design)
- PWA 支援離線存取
- 支援 PC / iOS / Android 瀏覽器

## 功能模組

### 1. 題目系統
- TOEIC Part 1-7 題目分類
- 難度分級 (Easy/Medium/Hard)
- 標籤系統

### 2. 學習系統
- 艾賓浩斯遺忘曲線複習
- 個人進度追蹤
- 答題歷史統計

### 3. 聽力系統
- TTS 語音播放
- 播放速度調整

### 4. 考試模擬
- 計時測驗模式
- 分數計算與分析
